Dashboard for Building Repair Forecasting
- Historic England

Power BI dashboard development for internal use at Historic England,
to forecast future building labour demands.

Project Overview: I developed a Power BI dashboard for internal use at Historic England, to forecast future building labour demands based on existing data about the age and type of buildings in each local authority. The dashboard provides visual insights into the number of older buildings across regions, allowing stakeholders to estimate future labour needs for maintenance or renovation, and economic output caused if these needs are met.

Key Features:
• Visualised Building Age Distribution: Displays the age profile of buildings across local authorities, highlighting areas with a higher proportion of older buildings.
• Building Type Insights: Breaks down the data by building type (residential, commercial, etc.), showing the variety of projects and skills that would need to be met in order to carry out necessary maintenance.
• Forecasting Labour Needs: The dataset takes into account the age and type of buildings to predict future labour requirements for older buildings that are likely to need renovation or replacement.

Technologies Used
• Power BI: For creating interactive data visualisations and generating insights from existing datasets.

Impact and Results
• The creation of this dashboard meant that our team was able to easily access building and labour needs figures relating to specific local authorities, even if they came from a very non-technical background.
